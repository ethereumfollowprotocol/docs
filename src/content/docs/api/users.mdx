---
title: Users
description: Ethereum Follow Protocol API v1
---

:::note

This is currently a test API with fake data and will receive updates with additional functionality over time. It is intended to be used to test integrations in preparation for **EFP**'s mainnet launch.

:::

API URL: [`https://api.ethfollow.xyz/api/v1`](https://api.ethfollow.xyz/api/v1)

## Routes

```sh
GET /:addressOrENS/followers
GET /:addressOrENS/following
GET /:addressOrENS/list-records
GET /:addressOrENS/primary-list
GET /:addressOrENS/stats
```

```

// sample response

}
```

### /users/\:addressOrENS/stats

Get number of followers and following by Address or ENS Name

```sh
curl http://api.ethfollow.xyz/api/v1/users/dr3a.eth/stats
```

```jsonc
// sample response
{
  "followers_count": 35,
  "following_count": 4
}
```
