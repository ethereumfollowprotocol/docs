#### /exportState/\:token_id
Get all accounts that are being followed by EFP list id, excludes blocks and mutes

#### Path Parameters
- `token_id` (string): The EFP List of the account

#### Sample Query
```sh
curl https://api.ethfollow.xyz/api/v1/exportState/333
```
