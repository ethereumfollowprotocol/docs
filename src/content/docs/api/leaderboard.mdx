title: Leaderboard
description: Ethereum Follow Protocol API v1
---

:::note

This is currently a test API with fake data and will receive updates with additional functionality over time. It is intended to be used to test integrations in preparation for **EFP**'s mainnet launch.

:::

API URL: [`https://api.ethfollow.xyz/api/v1`](https://api.ethfollow.xyz/api/v1)

# Routes

```sh
GET /leaderboard/followers?limit=10
GET /leaderboard/following?limit=10
```

## /leaderboard/followers

Get leaderboard - most followers

```sh
curl https://api.ethfollow.xyz/api/v1/leaderboard/followers?limit=3
```

```jsonc
// sample response
[
  {
    "rank": 1,
    "address": "0x0000000000000000000000000000000000000000",
    "followers_count": "49"
  },
  {
    "rank": 2,
    "address": "0x0000000000000000000000000000000000000001",
    "followers_count": "48"
  },
  {
    "rank": 3,
    "address": "0x0000000000000000000000000000000000000002",
    "followers_count": "47"
  }
]
```
