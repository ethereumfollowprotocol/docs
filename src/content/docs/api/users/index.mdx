:::note

This is currently a test API with fake data and will receive updates with additional functionality over time. It is intended to be used to test integrations in preparation for **EFP**'s mainnet launch.

:::

API URL: [`https://api.ethfollow.xyz/api/v1`](https://api.ethfollow.xyz/api/v1)

### Available Endpoints

```sh
GET /:ensOrAddress/followers
GET /:ensOrAddress/following
GET /:ensOrAddress/primary-list
GET /:ensOrAddress/stats
```

#### Get followers by ENS name or address

```sh
curl --request 'GET' \
  http://api.ethfollow.xyz/api/v1/users/dr3a.eth/followers
```

```jsonc
// sample response
{
  "followers": [
    {
      "follower": "0x0483499fac8c6eaccd64c5d316762af172f5eb4a",
      "tags": ["dr3a"]
    },

      "follower": "0x2a3bb9f9b1a305f39c917b8a5b7fba0be1fe3601",
      "tags": []
    },
    {
      "follower": "0x2de24a3a593ad27aa7f7db4a4a6204a57bf75d0b",

}
```

#### Get primary list by ENS name or address

```sh
curl --request 'GET' \
  http://api.ethfollow.xyz/api/v1/users/dr3a.eth/primary-list
```

```jsonc
// sample response
{
  "primary_list": "0"
}
```

#### Get number of followers and following by ENS name or address

```sh
curl --request 'GET' \
  http://api.ethfollow.xyz/api/v1/users/dr3a.eth/stats
```

```jsonc
// sample response
{
  "followers_count": 35,
  "following_count": 4
}
```
