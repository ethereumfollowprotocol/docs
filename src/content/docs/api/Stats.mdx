#### /stats
Get global EFP statistics. 

#### Sample Query
```sh
curl https://api.ethfollow.xyz/api/v1/stats
```

```jsonc
// sample response
{
  "stats": {
    "address_count": "8999",
    "list_count": "733",
    "list_op_count": "50284"
  }
}
```
