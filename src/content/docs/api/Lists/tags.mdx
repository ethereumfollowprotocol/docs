#### /lists/\:token_id/tags
Get the tags of a user by their EFP list id. 

#### Path Parameters
- `token_id` (string): The EFP List of the account

#### Sample Query
```sh
curl https://api.ethfollow.xyz/api/v1/lists/3/tags
```
