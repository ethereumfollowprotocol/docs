## Endpoint
`GET /:addressOrENS/account`

## Parameters

### Path Parameters
- `addressOrENS` (string): The address or ENS name of the account.

### Query Parameters
- `include` (string, optional): Specifies additional data to include in the response. Possible value:
  - `full`: Includes the primary list of the user.

## Response

### Success Response
- **Status Code**: `200 OK`
- **Content**:
  ```json
  {
    "address": "string",
    "ens": {
      "name": "string",
      "avatar": "string",
      "description": "string",
      "url": "string",
      "email": "string",
      "location": "string",
      "twitter": "string",
      "github": "string"
    },
    "primary_list": "string" // Only included if `include=full` is specified
  }
  ```

### Error Response
- **Status Code**: `404 Not Found`
- **Content**:
  ```json
  {
    "response": "ENS name not valid or does not exist"
  }
  ```

## Example Request
```bash
curl -X GET "https://api.example.com/0x1234567890abcdef/account?include=full"
```
