### /users/\:addressOrENS/ens
Get the ENS data of a user by their address or ENS name. 

#### Path Parameters
- `addressOrENS` (string): The address or ENS name of the account.

#### Sample Query
```sh
curl https://api.ethfollow.xyz/api/v1/users/brantly.eth/ens
```
