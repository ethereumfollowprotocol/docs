### /users/\:addressOrENS/stats
Get stats of a user by their address or ENS name. 

#### Path Parameters
- `addressOrENS` (string): The address or ENS name of the account.

```sh
curl https://api.ethfollow.xyz/api/v1/users/limes.eth/stats
```

```jsonc
// sample response
{
  "followers_count": "104",
  "following_count": "26"
}
```
