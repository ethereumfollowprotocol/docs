### /users/\:addressOrENS/lists
Get the lists of a user by their address or ENS name. 

#### Path Parameters
- `addressOrENS` (string): The address or ENS name of the account.

```sh
curl https://api.ethfollow.xyz/api/v1/users/brantly.eth/lists
```

```jsonc
// sample response
{
  "primary_list": "4",
  "lists": [
    "4",
    "107"
  ]
}
```
