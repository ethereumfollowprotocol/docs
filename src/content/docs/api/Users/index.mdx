API URL: [`https://api.ethfollow.xyz/api/v1`](https://api.ethfollow.xyz/api/v1)

## Routes

```sh
GET /users/:addressOrENS/followers
GET /users/:addressOrENS/following
GET /users/:addressOrENS/list-records
GET /users/:addressOrENS/primary-list
GET /users/:addressOrENS/stats
```

### /users/\:addressOrENS/primary-list

Get primary list by Address or ENS Name

```sh
curl http://api.ethfollow.xyz/api/v1/users/dr3a.eth/primary-list
```

```jsonc
// sample response
{
  "primary_list": "0"
}
```

### /users/\:addressOrENS/stats

Get number of followers and following by Address or ENS Name

```sh
curl http://api.ethfollow.xyz/api/v1/users/dr3a.eth/stats
```

```jsonc
// sample response
{
  "followers_count": 35,
  "following_count": 4
}
```
