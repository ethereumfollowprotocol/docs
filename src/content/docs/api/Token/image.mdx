#### /token/image/\:token_id
Get NFT image for a specified token id

#### Sample Query
```sh
curl https://api.ethfollow.xyz/api/v1/token/image/3
```

#### Response
This endpoint responds with a formatted svg of content-type `image/svg+xml;charset=utf-8`
