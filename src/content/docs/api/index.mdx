---
title: API (v1)
description: Ethereum Follow Protocol API
---

:::note

This is currently a test API with fake data and will receive updates with additional functionality over time. It is intended to be used to test integrations in preparation for EFP's mainnet launch.

:::

API URL: `https://api.ethfollow.xyz/v1`

### Available Endpoints

```sh
GET /stats/:address
GET /followers/:address
GET /following/:address
```

:::caution

Address must be [checksummed](https://eips.ethereum.org/EIPS/eip-55). You can use [viem](https://viem.sh/docs/utilities/getAddress.html#usage) or [ethers](https://docs.ethers.org/v5/api/utils/address/#utils-getAddress) to checksum an address.

:::

#### Get number of followers and following

```sh
curl --request 'GET' \
  https://api.ethfollow.xyz/v1/stats/0xeb6b293E9bB1d71240953c8306aD2c8aC523516a
```

```jsonc
// sample response
{
  "data": {
    "followersCount": 50,
    "followingCount": 3
  }
}
```

#### Get address followers

```sh
curl --request 'GET' \
  https://api.ethfollow.xyz/v1/followers/0xeb6b293E9bB1d71240953c8306aD2c8aC523516a
```

```jsonc
// sample response
{
  "data": [
    {
      "actor_address": "0x0483499Fac8C6EaCcD64c5D316762af172f5Eb4A",
      "action_timestamp": "2023-11-02T11:58:15+00:00"
    }
  ]
}
```

```sh
curl --request 'GET' \
  https://api.ethfollow.xyz/v1/following/0xeb6b293E9bB1d71240953c8306aD2c8aC523516a
```
